2 Setting up

Before starting, we need some sample data, and make sure out data folder is set up properly. The file weâ€™re using is a simple CSV file we download using he download.file() function.

We will also download the packages we need. Especially tidyverse which contains a large number of functions to make data management easier.

dir.create("data")

download.file(
  "https://raw.githubusercontent.com/datacarpentry/r-socialsci/main/episodes/data/SAFI_clean.csv",
  "data/SAFI_clean.csv", mode = "wb"
)

install.packages(c("tidyverse","here","arsenal","lmtest","sandwich","here",
                   "huxtable","flextable"))

Then we load the file in R, using the read_csv() function from the tidyverse package

library(tidyverse)

interviews <- read_csv("data/SAFI_clean.csv", 
  na = "NULL")
